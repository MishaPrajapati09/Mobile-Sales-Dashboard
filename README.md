# Mobile Sales Dashboard

## Project Overview
Interactive Power BI dashboard for analyzing mobile phone sales performance across cities, brands, payment methods, and customer ratings.

## Key Insights
- Total Sales Analysis
- Sales by City
- Brand Performance
- Customer Ratings Analysis
- Payment Method Distribution
- Daily Sales Trends

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling
